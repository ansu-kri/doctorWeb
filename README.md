<h1 align="center">🏥 Doctor Appointment & Healthcare Management System</h1>

<p align="center">
  A Modern Full-Stack Healthcare Platform built with React, Redux Toolkit, Node.js & MongoDB
</p>

This platform enables patients to securely authenticate, book appointments, and manage healthcare interactions, while providing scalable backend architecture for real-world deployment.

🚀 Live Features
🔐 Authentication System

Email + OTP Login

Google OAuth Login

Secure JWT-based authentication

Redux Toolkit state management

Protected routes

👨‍⚕️ Patient Features

Register / Login

Book doctor appointments

View appointment history

Responsive mobile-first design

🩺 Doctor Features (Scalable Module)

Doctor profile management

Availability scheduling

Appointment tracking

🛠 Backend Features

RESTful API architecture

MVC folder structure

Secure file uploads (uploads folder)

Environment-based configuration

MongoDB integration

Error handling middleware

🏗 Tech Stack
Frontend

⚛ React (Vite)

🧠 Redux Toolkit (State Management)

🎨 Tailwind CSS (Modern UI)

🔐 Firebase Google Authentication

📦 Axios (API handling)

🌐 React Router DOM

Frontend Setup
cd frontend
npm install
npm run dev

Backend

🟢 Node.js

🚀 Express.js

🍃 MongoDB + Mongoose

🔐 JWT Authentication

📁 Multer (File Uploads)

🔑 dotenv (Environment variables)

 Backend Setup
cd backend
npm install
npm run dev
